== Description ==

This unique Wordpress plugin adds a virtual room on your website, where your users can configure the virtual room directly with included devices.

Easy to add the highlightable areas of your image â€“ and easy to edit the devices/images later too

= Plugin features: =

** Backend: **
- Allow manage virtual room
- Allow manage all device in virtual room
- Manage device category wise

** Frontend: **
- User can select devices for virtual room, then they can see the overview of room with selected devices.

== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/ directory of your WordPress site.
3. Activate the plugin on the Plugin dashboard.

== Frequently Asked Questions ==

= Can I manage multi virtual room? =

Yes! You can manage multi room and assign device for them.

= Can I assign an device for many rooms ? =

Yes! In backend, you can do that when updating room detail. 

= Can I set multi or single selected device for each category? =

Yes! In backend, you can set category type is Checkbox or Option
